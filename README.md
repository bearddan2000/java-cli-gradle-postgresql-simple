# java-cli-gradle-postgresql-simple

## Description
A java gradle build, that connects to postgresql database.

## Tech stack
- java
- gradle
  - 6.8.2

## Docker stack
- postgresql:alpine
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
